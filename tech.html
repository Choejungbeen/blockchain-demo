사용된 기술

1. HTML

본 웹사이트는 다음과 같은 HTML 기술을 활용했습니다:

  • 시맨틱 태그: header, nav, main, section, footer 등을 사용하여 명확한 문서 구조를 구현했습니다.
  • HTML5 폼 요소: input, button 등을 활용하여 Demo 페이지의 대화형 요소를 구현했습니다.
  • 테이블: Topic 페이지에서 블록체인의 장단점을 비교하는 표를 제작했습니다.
  • 이미지 및 캡션: figure와 figcaption의 대안으로 div와 p class="caption"을 사용하여 블록체인 다이어그램에 설명을 추가했습니다.

2. CSS

다양한 CSS 기법으로 반응형, 사용자 친화적 디자인을 구현했습니다:

  • Grid 레이아웃: display: grid를 활용하여 팀원 소개와 블록체인 활용 사례를 격자 형태로 배치했습니다. 예시 코드: 
    
.team-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 2rem;
}
  • Flexbox: 헤더의 네비게이션 바와 Demo 페이지의 컨트롤 요소 배치에 활용했습니다.
  • 미디어 쿼리: 다양한 화면 크기에 맞게 레이아웃을 최적화했습니다. 예시 코드: 
    
@media (max-width: 768px) {
  .nav-links {
    display: none;
  }
  .hamburger {
    display: block;
  }
  .nav-links.active {
    display: flex;
    flex-direction: column;
    position: absolute;
    top: 60px;
    left: 0;
    width: 100%;
    background: #ffffff;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
  }
}
  • 애니메이션 및 전환 효과: 버튼 호버 효과, 메뉴 토글 시 부드러운 전환, Demo 페이지의 진행 바 애니메이션을 구현했습니다.
  • CSS 변수: 일관된 색상 테마와 간편한 유지보수를 위해 사용했습니다.

3. JavaScript

다양한 인터랙티브 기능을 JavaScript로 구현했습니다:

3.1. 반응형 네비게이션

모든 페이지에서 햄버거 메뉴 토글 기능을 구현했습니다:


const hamburger = document.querySelector(".hamburger");
const navLinks = document.querySelector(".nav-links");
hamburger.addEventListener("click", () => {
  navLinks.classList.toggle("active");
});

3.2. 블록체인 시뮬레이션

Demo 페이지에서는 실제 블록체인의 핵심 원리를 구현한 JavaScript 기능을 제공합니다:

  • SHA-256 해싱: CryptoJS 라이브러리를 활용하여 블록 데이터의 암호화 해시를 계산합니다: 
    
function calculateHash(index, data, prevHash, nonce) {
  return CryptoJS.SHA256(index + data + prevHash + nonce).toString();
}
  • 작업 증명(PoW) 시뮬레이션: 특정 난이도('00'으로 시작하는 해시)를 만족하는 nonce 값을 찾는 마이닝 과정을 시뮬레이션합니다: 
    
function mineStep() {
  if (!mining) return;

  pendingBlock.nonce = nonce;
  pendingBlock.hash = calculateHash(pendingBlock.index, pendingBlock.data, pendingBlock.prevHash, nonce);

  progress = Math.min((nonce / 1000) * 100, 100);
  document.getElementById("progress").style.width = `${progress}%`;

  if (pendingBlock.hash.startsWith(difficulty)) {
    // 채굴 성공 로직
    blockchain.push(pendingBlock);
    // ...
  } else if (nonce >= 1000) {
    // 채굴 시간 초과 로직
    // ...
  } else {
    nonce++;
    setTimeout(mineStep, 50); // 비동기 실행으로 UI 블로킹 방지
  }
}
  • 진행 상태 시각화: 마이닝 진행 상황을 실시간으로 보여주는 프로그레스 바를 구현했습니다.
  • 블록체인 렌더링: 생성된 블록을 시각적으로 표현하는 기능을 구현했습니다: 
    
function renderBlockchain() {
  const container = document.getElementById("blockchain");
  container.innerHTML = "";
  blockchain.forEach((block) => {
    const blockElement = document.createElement("div");
    blockElement.className = "block";
    blockElement.innerHTML = `
      
    블록 #${block.index}
    
      
    데이터: ${block.data}
    
      
    이전 해시: ${block.prevHash.substring(0, 16)}...
    
      
    Nonce: ${block.nonce}
    
      
    해시: ${block.hash.substring(0, 16)}...
    
    `;
    container.appendChild(blockElement);
  });
}
